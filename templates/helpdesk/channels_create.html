Создание канала

  1. Зарегистрируйтесь на сайте и войдите в панель.
  2. На главной странице нажмите на карточку «+ Создать канал»
  3. Придумайте название для вашего канала. По этому названию канал можно будет определить в панели
  4. При успешном создании вы снова окажетесь на главной странице

Карточка канала выглядит так:
{% include 'helpdesk/card.html' %}

На ней отображаются:

  • Название канала, введённое вами ранее
  • #ID канала, выданный системой
  • Статус активности, канал активен, если есть хотя бы один активный ключ
  • Индикаторы количества ключей доступа: R - количество ключей на чтение, W - на запись
  • Кнопка для настройки канала
