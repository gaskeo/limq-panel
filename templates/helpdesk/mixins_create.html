Создание миксинов

  1. Пройдите в настройки нужного канала
  2. Перейдите в пункт меню «Миксины»

Страница настроек миксинов выглядит так:

{% with tab='mixins' %} {% include 'helpdesk/edit_menu/edit_menu.html' %} {% endwith %}

  1. Введите ключ на чтение другого канала

    Ключ должен быть с другого канала. Канал нельзя подписать на самого себя

  2. При успешном создании миксин добавится в список под формой в меню «Входящие». В меню «Исходящие» миксины, которые подписаны на вас

Карточка миксина выглядит так:

{% include 'helpdesk/edit_menu/mixin.html' %}

На ней отображаются:

  • Название канала, на который вы подписаны или который подписан на вас
  • ID канала, на который вы подписаны или который подписан на вас
  • Кнопка для удаления миксина
