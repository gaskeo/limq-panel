Создание ключа

  1. Пройдите в настройки нужного канала
  2. Перейдите в пункт меню «Ключи доступа»

Страница настроек ключей выглядит так:

{% with tab='keys' %} {% include 'helpdesk/edit_menu/edit_menu.html' %} {% endwith %}
  1. Придумайте название для вашего ключа. По этому названию ключ можно будет определить
  2. Выберите права для этого ключа: на прием или отправку
  3. При успешном создании ключ добавится в список под формой

Карточка ключа выглядит так:

{% include 'helpdesk/edit_menu/key.html' %}

На ней отображаются:

  • Название ключа, введённое вами ранее
  • ID ключа, выданный системой
  • Кнопки для приостановки и удаления ключа
