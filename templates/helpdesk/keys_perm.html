Выбор прав доступа

Вы можете выбрать для ключа права доступа:

  1. На прием.
    Данное право доступа позволяет принимать сообщения с канала, которые поступают посредством других ключей.
    При выборе этого права открывается дополнительный пункт «Разрешить info». С помощью данного чекбокса можно настроить отбражение миксинов для ключа в статистике.
    Если ключ публичный, рекомендуется не ставить этот чекбокс
  2. На отправку.
    С помощью такого ключа можно отправлять сообщения на канал
